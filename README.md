# Retail Sales Analysis using SQL

## 📌 Project Objective
Analyze retail sales data to identify revenue trends, top-performing products, and regional performance.

## 🛠 Tools Used
- MySQL
- Excel

## 📊 Key Analysis Performed
1. Monthly Revenue Trend
2. Top 10 Products by Sales
3. Region-wise Sales Performance
4. Customer Purchase Frequency

## 🧠 SQL Concepts Used
- SELECT, WHERE
- GROUP BY
- HAVING
- INNER JOIN
- Subqueries
- Aggregate Functions

## 📈 Business Insights
- Identified peak revenue months
- Found top-selling products
- Highlighted underperforming regions
